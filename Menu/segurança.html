Comandos do Windows

Use os comandos nativos do Windows para manutenção, diagnóstico e otimização do seu computador.
Clique em uma das opções abaixo para ver exemplos de comandos.

  • CMD (Prompt de Comando)
    CMD permite executar comandos básicos do Windows para manutenção e diagnóstico.
  • PowerShell
    PowerShell é uma ferramenta avançada para automação, scripts e segurança no Windows.
  • Windows + R
    O atalho Windows + R abre a caixa de diálogo "Executar" para iniciar rapidamente programas e ferramentas.

Segurança no Windows

1. Antivírus e Firewall

O que é?

  • Antivírus é um programa que verifica se há vírus ou arquivos perigosos no seu computador e os bloqueia.
  • Firewall é como uma muralha que protege seu computador de acessos estranhos pela internet.

O que fazer?

  • Use o antivírus que já vem com o Windows, chamado Windows Defender. Ele protege automaticamente seu computador.
  • Verifique se ele está sempre ligado e atualizado para identificar as ameaças mais recentes.

2. Atualizações

O que são?

Atualizações são melhorias enviadas pela Microsoft para corrigir problemas e deixar o sistema mais seguro.

O que fazer?

  • Sempre que aparecer uma mensagem para atualizar o Windows, aceite e instale.
  • Atualize também os programas que você usa e os drivers (programas que fazem o hardware funcionar direito).

3. Navegação Segura

O que é?

Navegar com cuidado evita que você acesse sites perigosos que podem roubar seus dados.

O que fazer?

  • Não clique em links estranhos que chegam por email ou mensagens.
  • Antes de colocar qualquer informação pessoal, verifique se o site tem “https://” e um cadeado na barra do navegador — isso significa que o site é seguro.

4. Backup Regular

O que é?

Backup é uma cópia dos seus arquivos importantes. Assim, se o computador der problema, você não perde tudo.

O que fazer?

  • Salve seus documentos, fotos e vídeos importantes num HD externo (um dispositivo que você conecta ao computador) ou na nuvem (Google Drive, OneDrive).
  • Faça backup sempre que puder para manter seus arquivos seguros.
